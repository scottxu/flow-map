##流向地图
流向地图常用来可视化源汇流(Origin Destination Flow)数据。源地和汇地可以是点，也可以是面。源地和汇地之间的互动数据，常用连接空间单元几何重心之间的线段来表达，线的宽度或颜色来表示源地和汇地之间的流向数值。每个空间位置可以是源地，也可以是汇地，因此必要时，线段上需要用箭头来表示流向方向

##应用场景

* 流向地图多应用于区际贸易
* 交通流向
* 人口迁移
* 购物消费行为
* 通讯信息流动
* 航空线路。
